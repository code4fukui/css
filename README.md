# CSS

CSS project that provides a rainbow text effect.

## Demo
[rainbow demo](https://code4fukui.github.io/css/rainbow.html)

## Features
- Animated rainbow text effect
- Based on CSS gradient and animation

## Usage
To use the rainbow effect, include the CSS file and apply the `rainbow` class to the desired text:

```html
<link rel="stylesheet" type="text/css" href="https://code4fukui.github.io/css/rainbow.css">
<div class="rainbow">rainbow</div>
```
