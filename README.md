# CSS

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A lightweight CSS library for creating an animated rainbow text effect with a blue glossy glow.

## Demo

[View the live demo](https://code4fukui.github.io/css/rainbow.html)

## Features

-   Animated rainbow gradient background using `linear-gradient` and `@keyframes`.
-   White text with a blue glossy `text-shadow` effect.
-   Lightweight and dependency-free.

## Usage

Include the CSS file and apply the `rainbow` class to any HTML element.

```html
<link rel="stylesheet" type="text/css" href="https://code4fukui.github.io/css/rainbow.css">

<h1 class="rainbow">Rainbow Text</h1>
```
