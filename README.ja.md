# CSS

青く光沢のあるグロー効果を伴う、アニメーションする虹色テキストを作成するための軽量CSSライブラリです。

## デモ

[ライブデモを見る](https://code4fukui.github.io/css/rainbow.html)

## 特徴

- `linear-gradient` と `@keyframes` を使用したアニメーションする虹色グラデーション背景。
- 青く光沢のある `text-shadow` 効果を備えた白いテキスト。
- 軽量で依存関係なし。

## 使い方

CSSファイルを読み込み、任意のHTML要素に `rainbow` クラスを適用してください。

```html
<link rel="stylesheet" type="text/css" href="https://code4fukui.github.io/css/rainbow.css">

<h1 class="rainbow">Rainbow Text</h1>
```
